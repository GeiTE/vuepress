# 前端开发

## 前端开发岗位		

​		前端开发我现阶段的理解，工作位置定义在`UI`后，测试前，与后端开发主要配合的位置。工作内容是项目前期的基本设计讨论、与UI和后端沟通获取可行的`UI`设计方案和后端接口文档，完成前端项目的开发，部署至应用服务器中。

​		根据不同的公司情况，前端工程师（很大一部分中小微或非撰写互联网公司前端工程师岗）可能会兼职项目前期策划工作、`UI`工作、测试工作、运维工作。

​		互联网行业应用开发方向前端岗位，现在（2023年10月）进入了初保中缺高缺（初级前端开发区域饱和平稳，中高级前端开发工程师任然缺少）的一种就业环境，因此你会发现刚毕业出来的参与前端开发的薪资被压得很低（很多技术岗位都是如此），两年至三年开发经验充足，进入中级开发行列，薪资能够提升到`8k+`。

​		前端开发时互联网的一种生态，物联网的持续普及和发展，前端开发将会不断的有需求，有需求的同时，中高级前端开发工程师开发的低代码工具在给底层程序员提供便利的同时，也在挤压底层程序员的价值和生存空间。逐渐的底层程序员会被美工抢占，而程序员的入门门槛进入到低代码工具维护升级的新阶段。
